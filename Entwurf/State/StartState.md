# Start State

## Einleitung

Der Start state ist der State in den der Koordinator beim Starten ist.
In diesem State registriert sich der Koordinator und bereitet alles vor.

## Detailierte Beschreibung

Im Start state registriert sich der Koordinator beim Namensdienst mit seinem Namen.

Danach wechselt er automatisch in den init State, wo er dann Anfragen entgegen nehmen kann.
